SpiceWheel

A templating library for image grids in React

Why SpiceWheel?

Responsive image-grids are a common feature in web applications. SpiceWheel makes it easy to rapidly prototype these components using pre-defined styles and a simple interface.

Spice up your front-end!

Increase productivity with SpiceWheel

How does SpiceWheel work?

SpiceWheel is an opinionated library. If you want to use it, you need to do something that no modern front-end web developer has ever done, use HTML tables. At SpiceWheel, we prefer to call these beautiful relics SpiceRacks.

The SpiceWheel React component parses each row of a SpiceRack and converts each row into a list element of an unordered list. While the list is being constructed, SpiceWheel is injecting the style you've selected into each of the components of the list.

SpiceWheel was built to be modular and open to the community. Feel free to submit style suggestions or create your own! For infromation about how to get involved, please visit our GitHub repo.
